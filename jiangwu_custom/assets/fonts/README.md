# 字体文件占位符

此目录用于存放项目使用的字体文件。

## 需要的字体文件：
- NotoSerifSC-Bold.otf - 标题字体（思源宋体）
- NotoSansSC-Regular.otf - 正文字体（思源黑体）

## 字体规格：
- 格式：OTF (OpenType Font)
- 编码：UTF-8
- 包含：简体中文、英文、数字

## 下载地址：
- 思源宋体：https://github.com/adobe-fonts/source-han-serif/releases
- 思源黑体：https://github.com/adobe-fonts/source-han-sans/releases

## 注意事项：
1. 字体文件较大（约10-20MB）
2. 建议使用子集化工具减小体积
3. 确保字体许可证允许商用
