# 匠物定制 - 小众手作艺术品定制APP

> 一款基于Flutter+RAG+多模态理解的小众手作艺术品定制应用，覆盖用户需求发布、3D实时预览、定制交易全流程闭环、AI辅助设计、溯源存证等核心能力。

---

## 一、功能设计

### 1.1 核心功能

#### 用户端功能

| 功能模块 | 功能说明 | 业务规则 | 优先级 |
|---------|---------|---------|--------|
| **多模态需求发布** | 用户上传参考图、文字描述、语音说明，系统自动解析生成定制需求单 | 支持JPG/PNG格式，单图≤10MB；文字≤500字；语音≤60秒 | P0 |
| **3D实时预览** | 用户调整材质、尺寸、颜色、雕刻内容后，实时渲染3D模型 | 支持旋转360°、缩放0.5x-3x、多角度截图保存 | P0 |
| **智能推荐** | 基于用户历史浏览、风格偏好，推荐匹配的手作人和工艺方案 | 推荐权重：风格匹配40%、手作人评分30%、交期匹配20%、价格区间10% | P1 |
| **定制交易** | 需求发布→报价→签订→支付分担保→阶段交付→确认→评价 | 支付分担保期限：最长90天；确认超时：7天自动推进下一阶段 | P0 |
| **溯源存证** | 订单信息、定制参数、交付图片上传联盟链 | 存证时机：订单签订、阶段交付、最终交付、评价完成 | P2 |

#### 手作人端功能

| 功能模块 | 功能说明 | 业务规则 | 优先级 |
|---------|---------|---------|--------|
| **工艺图上传** | 手作人上传工艺图、材质说明、工艺参数 | 支持批量上传，最多20张；工艺参数支持JSON格式导入 | P0 |
| **智能报价** | 系统根据工艺难度、材质成本、工时自动计算建议报价 | 手作人可调整报价，但不得低于系统建议价的70% | P1 |
| **订单管理** | 接单、阶段交付、确认收货、评价管理 | 阶段交付提醒：到期前3天推送通知 | P0 |
| **作品展示** | 手作人主页、作品集、工艺详情展示 | 支持视频展示，时长≤5分钟 | P1 |

#### 管理后台功能

| 功能模块 | 功能说明 | 优先级 |
|---------|---------|--------|
| **用户管理** | 用户信息、认证状态、信用评级管理 | P1 |
| **订单监控** | 订单状态实时监控、异常订单预警、争议订单仲裁 | P1 |
| **内容审核** | 图片/文字/语音内容审核、敏感词过滤 | P2 |
| **数据统计** | 交易数据、用户行为、手作人业绩统计分析 | P2 |

### 1.2 拓展功能

| 功能模块 | 功能说明 | 优先级 |
|---------|---------|--------|
| **AI辅助设计** | 基于参考图自动拆分设计元素、匹配手作风格、推荐适配工艺 | P2 |
| **风格迁移** | 用户上传普通图片，AI自动转换为手绘/油画/水彩等艺术风格 | P3 |
| **语音定制** | 用户通过语音描述需求，系统自动转文字并解析定制参数 | P3 |
| **AR预览** | 支持AR查看定制成品在实际环境中的效果（预留接口） | P3 |

### 4.3 多模态端侧RAG方案

**技术选型：**
- 端侧推理：TensorFlow Lite + Qwen-VL量化模型
- 向量检索：FAISS本地索引 + 云端Milvus同步
- 知识库：MySQL存储结构化数据 + OSS存储非结构化数据

**实现流程：**
```
用户上传图片/文字
    ↓
端侧图像分割（TFLite U-Net）
    ↓
特征提取 + 向量化（CLIP模型）
    ↓
本地FAISS检索相似案例（Top-K）
    ↓
云端RAG增强（LangChain + Qwen-VL）
    ↓
返回定制建议（材质/工艺/手作人推荐）
```

**性能优化：**
- 模型量化：FP32→INT8，模型体积减少75%
- 图像预处理：端侧完成，减少网络传输
- 结果缓存：LRU缓存最近1000次检索结果
- 并发推理：异步处理多张图片

### 4.4 3D实时预览方案

**技术选型：**
- 渲染引擎：flutter_unity_widget + Three.js
- 模型格式：glTF 2.0（支持PBR材质）
- 轻量化：Draco压缩 + LOD多级细节

**渲染流程：**
```
用户调整参数（材质/尺寸/颜色/雕刻内容）
    ↓
参数序列化为JSON配置
    ↓
调用3D渲染引擎（Unity/Three.js）
    ↓
实时渲染预览模型（支持PBR材质）
    ↓
支持截图/录屏保存（导出PNG/MP4）
```

**性能优化：**
- 模型轻量化：Draco压缩，面数减少80%
- 纹理压缩：ASTC格式，内存占用减少50%
- 渲染优化：脏标记机制，仅重置变化部分
- GPU加速：Metal/Vulkan后端
